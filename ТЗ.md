# 11. 📌 Пример кода (с комментариями)

## Controller

```java
// Контроллер для работы с угрозами
@RestController
@RequestMapping("/threats")
public class ThreatController {

    private final ThreatService threatService;

    public ThreatController(ThreatService threatService) {
        this.threatService = threatService;
    }

    // Получить список всех угроз
    @GetMapping
    public List<Threat> getAllThreats() {
        return threatService.getAllThreats();
    }
}
```

---

## Service

```java
// Сервисный слой для бизнес-логики
@Service
public class ThreatService {

    private final ThreatRepository threatRepository;

    public ThreatService(ThreatRepository threatRepository) {
        this.threatRepository = threatRepository;
    }

    // Получение всех угроз из базы данных
    public List<Threat> getAllThreats() {
        return threatRepository.findAll();
    }
}
```

---

## Entity

```java
// Сущность "Угроза"
@Entity
public class Threat {

    @Id
    @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;

    // Название угрозы
    private String name;

    // Описание угрозы
    private String description;

    // Вероятность
    private double probability;

    // Ущерб
    private double impact;
}
```
